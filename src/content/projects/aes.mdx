## El proyecto

Plataforma de educación sanitaria con blog, eventos, galería multimedia, directorio de especialistas y panel administrativo. Mi primer proyecto que llegó a producción.

## Mi rol (backend)

- API REST con Express + MongoDB
- Autenticación JWT con roles (superadmin, admin, editor)
- Validación con express-validator
- Subida de imágenes a Cloudinary
- Emails transaccionales con Brevo (Sendinblue)
- Seed de datos automático al iniciar

### Frontend (colaboración)

- React + Vite + Redux Toolkit
- Panel administrativo completo (CRUD de contenido)
- Integración con la API vía Axios

## Lo que aprendí

Fue mi primer proyecto en producción. Me enseñó que tener algo funcionando con usuarios reales vale más que mil proyectos de tutorial. También aprendí que la validación consistente y el manejo de errores centralizado son de lo primero que hay que tener en cualquier API.
